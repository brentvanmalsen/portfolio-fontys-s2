Advies Website Weldas Europe

Wat zijn mogelijke verbeteringen voor de Weldas Europe website op het gebied van UI en UX?

Advies voor verbeteringen

Ik schrijf een advies voor verbeteringen voor de website van Weldas Europe, dit advies schrijf ik door te kijken naar de huidige user experience en ook het huidige design met als doel om de user experience van de website te verbeteren. Ik gebruik een onderzoeksmethode genaamd 'Heuristische evaluatie', met dit onderzoek gebruik ik een lijst met beoordelingscriteria en zoek ik hiermee naar mogelijke verbeterpunten. Op basis van mijn bevindingen maak ik een advies waarin ik de verbeteringen die ik heb gevonden op een rij zet.

Resulaten en de conclusie

Ik ben vooral veel verbeterpunten tegengekomen op het gebied van UX. Zo heb je bijvoorbeeld bij de producten in de listerpagina een knop met 'PRODUCT DETAILS'. Het is hierbij voor mij niet meteen duidelijk dat ik hiermee op de productpagina kom. Verder is het erg lastig dat ik direct na het toevoegen van mijn eerste product aan mijn winkelwagen alle klantgegevens dien in te vullen zonder dat ik eerst verder kan kijken. Ook zijn er een aantal nogal cryptische termen op de website aanwezig waarvan ik verwacht dat niet iedere klant dit meteen gaat begrijpen, denk hierbij aan 'End user' en 'Exhibition S&S 2023'. Wat verder ook opvalt is dat er onderaan in de footer een knop zit met een URL naar een externe website waarvan de pagina niet meer gevonden kan worden. Dit waren een aantal van de grotere mogelijke verbeteringen die ik ben tegengekomen. Ik heb vooral verbeteringen gevonden op het gebied van user experience.
