Testplan voor de usability test

Ik maak een testplan voor het uivoeren van de usability test die ik uitvoer op de Weldas Europe website. Ik maak een testplan zodat ik precies weet hoe ik de test moet gaan uitvoeren wat ik hierbij ga vertellen. In het testplan neem ik op wat het testdoel is, bij wie ik een test ga uitvoeren, wat ik ga meten, hoe ik het ga meten en laat ik de scenario's zien. Ik vermeld in dit testplan ook een stappenplan die ik bij de hand kan houden tijdens het uitvoeren van de usability test.

Wat is een testplan?

Ik heb onderzocht wat een testplan is en waarom het goed en handig is om een testplan te maken. Een testplan is een document dat de aanpak beschrijft voor het testen van een specifiek product, systeem of softwaretoepassing. In een testplan worden de testdoelen en de benodigde middelen voor het uitvoeren van de test opgenomen. Het testplan is een deel van de gehele usability test, in de usability test zijn verder ook de resultaten en de conclusie opgenomen.
