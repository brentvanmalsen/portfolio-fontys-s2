User Test Branding Vertile

Welke verbeteringen kan ik nog maken in mijn ontwerpen en de website voor de artiest Vertile?

test Design en user experience

Ik voer een user test uit met betrekking op het logo, de merchandise en de webshop die ik voor de artiest Vertile heb gemaakt. Ik stel vragen aan mijn doelgroep met als doel erachter te komen welke verbeteringen ik nog kan maken. Buiten alleen het design om stel ik ook een aantal scenario's op die de testpersonen mogen doorlopen op de webshop. De vragen die ik stel gaan over het design van het logo, de merchandise en de website. De scenario's zijn voor verbeteringen in de user experience van de website. Ik vertel wat ik ga testen en maak hiervoor een stappenplan. Ik schrijf mijn bevindingen op en ik verwerk de input van de testpersonen in de producten.

Wat is de conclusie?

Ik heb veel goede input gekregen van de testpersonen, zowel op het gebied van design als de user experience. De hoogtepunten uit deze feedback zijn om mee te beginnen de roze rand rondom de naam "Vertile" op het "VERTILE DIMENSIO X SHIRT". Meerdere testpersonen vonden dit niet mooi bij het ontwerp passen en ook niet bij Vertile als artiest, dit heb ik aangepast naar de primaire kleur van de naam op deze T-shirts. Wat ik verder ook goede input vond was dit 1 testpersoon vertelde dat hij een witte versie van het koningsdag shirt mooier vond, hij vond dit niet alleen mooier maar wit trekt ook minder warmte aan op een warme dag. Op het gebied van user experience hadden 2 van de 3 testpersonen enige moeite met het vinden van de social media icoontjes, deze icoontjes heb ik opvallender gemaakt door de kleur te veranderen naar wit. Verder heb ik nog een "Terug" pijl gemaakt op de productpagina's omdat de meeste testpersonen het moeilijk vonden om terug te keren naar de listerpagina.
