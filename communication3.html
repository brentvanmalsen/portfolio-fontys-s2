Projectbegeleider voor Sprint X

Ik heb Paul gevraagd om mijn projectbegeleider voor Sprint X te zijn. Ik vertel in dit verslag waarom ik ervoor gekozen heb om Paul als mijn projectbegeleider te vragen. Ook benoem ik een aantal onderwerpen waar ik tot nu toe al feedback op heb gekregen, denk hierbij aan mijn plan van aanpak voor Sprint X. Ik had in mijn planning bijvoorbeeld nog niet gedacht aan het maken van een Figma design voor mijn merchandise pagina, verder had ik ook niets opgeschreven over het uitvoeren van een onderzoek waar ik mijn inspiratie voor de branding uit kan halen. Ik vertel welke ideeën ik heb gekregen.

Hulp bij mijn Sprint X idee

Ik had wat moeite met het kiezen wat ik voor Sprint X wilde gaan doen. Graag had ik iets met design gedaan maar tegelijkertijd wilde ik ergens voor kiezen wat goed zou aanvullen op mijn portfolio leerdoelen waar ik nog iets voor wilde maken. Na feedback gevraagd te hebben aan Anke heeft ze mij een aantal ideeën gegeven over hoe ik in Design nog steeds verschillende leerdoelen kan aantonen, denk hierbij aan het uitvoeren van een user test of het doen van onderzoek naar branding. Ik wilde namelijk het liefst iets met design doen waarbij ik wél aan verschillende leerdoelen kan werken. Ten slot heb ik verteld over het vragen van extra feedback op mijn portfolio website omdat het semester niet al te lang meer duurt.
