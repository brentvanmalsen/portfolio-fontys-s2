Vertile Merchandise Website

Development

Een webhop voor vertile

Voor mijn Sprint X project heb ik gekozen om een branding te maken voor mijn favoriete artiest. Ik heb gekozen om een branding te maken omdat ik design erg leuk vind, ook vind ik deze artiest heel vet en kan ik met deze proftaak veel verschillende leerdoelen aantonen. Voor development heb ik een webshop gemaakt met een listerpagina en een productpagina. Op deze website komen zowel het logo als de merchandise naar voren die ik voor Vertile heb gemaakt. Vertile heeft tot op de dag van vandaag zelf geen merchandise en ook geen website, ondanks dat hij behoort tot één van de grootste upcoming hardstyle/rawstyle artiesten. Ook hierom leek het mij leuk om dit zelf te gaan proberen.

Grootste leermomenten

    Van de volgende aspecten heb ik het meeste geleerd:

  • Ik heb nieuwe CSS eigenschappen van flexbox geleerd, zoals justify-content: space around & flex-direction: column
  • Ik heb geleerd wat position: relative doet en hoe ik mijn HTML elementen kan hiermee kan positioneren
  • Ik heb meer inzicht gekregen in de werking van parent en child elementen
  • Ik heb geleerd dat er een footer HTML tag is
