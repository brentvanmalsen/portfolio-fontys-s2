Contactpagina Weldas Europe

Tijdens de proftaak Development heb ik een nieuwe contactpagina voor de website van Weldas Europe geprogrammeerd. De contactpagina is gebaseerd op het ontwerp (van mijn nieuwe groepje) uit Figma. Ik heb de website mobile first gemaakt, achteraf heb ik door gebruik te maken van media queries ook de desktopversie van deze pagina gemaakt. Ik heb bij het maken van de pagina in de branch 'feature/productpagina' gewerkt.

Grootste leermomenten

    Van de volgende aspecten heb ik het meeste geleerd:

  • Ik heb geleerd wat media queries zijn en hoe ik ze kan toepassen om de website responsive te maken
  • Ik heb geleerd hoe ik een functie maak in JavaScript met een if / else if statement en hoe ik deze functie aanroep in HTML
  • Ik heb geleerd hoe ik een inline frame vanaf een andere website kan implementeren
  • Ik heb meer kennis gekregen over het gebruiken van flexbox en de bijbehorende eigenschappen zoals justify-content en align-items
